[h2]模组功能[/h2]

每天白天自动发布今日推荐食谱，食用推荐的食物可获得额外属性加成和特殊增益效果。支持原版食谱。

[hr][/hr]

[h2]效果与配置[/h2]

[b]属性加成[/b]
食用今日食谱时，生命、饥饿、理智恢复效果获得额外加成。
可选倍数：1.2倍 / 1.5倍 / 2.0倍 / 2.5倍 / 3.0倍

[b]额外Buff[/b]
[list]
[*] 加速：移动速度提升30%
[*] 发光：角色周围产生光照
[*] 吸血：攻击回复20%伤害的生命值
[/list]
持续时间：30秒 / 60秒 / 120秒 / 180秒 / 300秒

[b]配置选项[/b]
[list]
[*] 每日食谱数量：1-5个（默认3个）
[*] 加成倍数：1.2-3.0倍（默认1.5倍）
[*] 公告文字：今日推荐 / 每日特餐 / 今日菜单 / 厨师推荐
[*] 额外Buff：无 / 加速 / 发光 / 吸血
[*] Buff持续时间：30-300秒
[/list]
